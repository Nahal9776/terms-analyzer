# 📜 Terms & Conditions Analyzer

## 🔍 Overview
An intelligent tool that summarizes lengthy Terms & Conditions documents and highlights potentially risky sections.

## ❓ Problem
T&C documents are too long and complex — people skip reading them and unknowingly agree to privacy risks or hidden fees.

## 💡 Solution
This tool automatically extracts key phrases, summarizes long paragraphs, and flags danger zones like auto-renewals, data sharing, etc.

## 🛠️ Tech Stack
- Python
- spaCy or transformers
- Flask / Streamlit (for UI)
